```

    ↓

    ├─→ NLPJsIntegration.processIntent()
    │   ├─→ Intent Detection
    │   ├─→ Entity Extraction
    │   └─→ Sentiment Analysis
    ├─→ TransformersIntegration (Fallback/Enhancement)
    │   ├─→ Sentiment Analysis
    │   └─→ Entity Extraction
    ↓
UnifiedNLPResult
    ├─→ intent
    ├─→ entities
    ├─→ sentiment
    └─→ engine used
```

#### 2. Intent-Based Routing
```
UnifiedNLPResult
    ↓
UnifiedAIAgent.routeByIntent()
    ├─→ users.list → User Management
    ├─→ booking.create → Optimization Engine
    ├─→ rbac.check → Authentication Firewall
    ├─→ dashboard.view → Intelligence Engine
    └─→ forecast.* → Forecasting Engine
```

- Text Generation: `nlp:write`

**Forecasting Features**:
- Ensemble Forecasting: `forecasting:read`
- Trend Detection: `forecasting:read`

- Schedule Optimization: `optimization:write`
- Pricing Optimization: `optimization:write` + `pricing:write` (admin only)

**Clustering Features**:
- User Segmentation: `clustering:write`
- Ensemble Clustering: `clustering:write`

**Reasoning Features**:
- Intelligent Reasoning: `reasoning:read`
- Conversation Context: `reasoning:read`

### Authentication Levels ✅

- **Public**: No authentication required
- **Authenticated**: User must be logged in
- **Admin**: User must have admin role
- **System**: System-level access only

### Access Control Flow ✅

```
User Request
    ↓
AuthenticationFirewall.validate()
    ├─→ Check JWT Token
    ├─→ Verify User Role
    └─→ Check Permissions
    ↓
AuthenticatedAIManager.checkAccess()
    ├─→ Component Access Control
    ├─→ Permission Enforcement
    └─→ Audit Logging
    ↓
Engine Execution
    ↓
Response (with audit trail)
```
