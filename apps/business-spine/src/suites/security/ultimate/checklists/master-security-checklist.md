# Master Security Checklist (Authoritative)

This checklist represents the COMPLETE security surface for a modern online platform.

Each item must be:
- Implemented
- Documented
- Monitored
OR explicitly accepted as a risk

Failure to do so is a launch blocker.

Sections:
- Authentication & Authorization
- Application Security
- Data Protection & Encryption
- Network Security
- Infrastructure Security
- Secrets & Configuration Management
- Code Security
- Monitoring & Incident Response
- Compliance & Governance
- Physical & Operational Security
- Third-Party & Supply Chain
- Browser & Client-Side Security
- Testing & QA
- Backup & Recovery
- Documentation & Knowledge
- Advanced / Emerging Security

Refer to individual folders for enforceable controls and templates.
