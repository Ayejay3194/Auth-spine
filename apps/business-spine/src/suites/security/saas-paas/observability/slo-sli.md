# SLO/SLI starter set
SLIs:
- Availability (per region)
- API latency P50/P95/P99
- Error rate (5xx)
- Webhook delivery success %
- Queue processing latency

SLO examples:
- API availability: 99.9% monthly (Free), 99.95% (Business), 99.99% (Enterprise)
- P95 latency: < 500ms for core endpoints
- Webhook delivery: 99% success within 5 minutes
